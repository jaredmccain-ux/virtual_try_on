Virtual Try-On Style Editor

Instruction-Guided Image Editing with Multi-Model Comparison
Pattern Recognition Course Project · HUST 2026

Report Code Demo

Abstract

This project compares five instruction-guided image editing models for virtual try-on style editing: PromptDresser, Qwen-Image-Edit, FLUX.1-Kontext-dev, FireRed-Image-Edit, and DreamOmni2. Users upload a person image and a garment image, provide a text instruction describing the desired style change, and the system generates edited results for side-by-side comparison across models.

Model Comparison

Model Paradigm Key Feature Parameters
PromptDresser Diffusion + Prompt Enhancement Generative textual prompts with prompt-aware mask for controllable virtual try-on Not disclosed (diffusion-based)
Qwen-Image-Edit MMDiT Dual-Decoder Diffusion Semantic + appearance dual-mode editing; precise Chinese/English text rendering ~27B (MMDiT 20B + Qwen2.5-VL 7B)
FLUX.1-Kontext-dev Flow Matching Context Transformer Flow matching with in-context generation; strong character consistency & local editing 12B
FireRed-Image-Edit MMDiT High-Fidelity Editing High-fidelity editing with identity preservation, multi-element fusion & old photo restoration ~28B (Transformer 20B + Text Encoder ~8B)
DreamOmni2 MLLM Instruction + LoRA Tuning Multimodal instruction-driven unified editing & generation with multi-image input FLUX.1 Kontext + Qwen2.5-VL 7B LoRA
